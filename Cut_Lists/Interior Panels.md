# Interior Panels Cut List

This cut list will produce paneling for two homes. Get 3 orange carts
for this work.

## Horizontal Cuts

Set up the panel saw for horizontal cuts. Using the templates, cut in
this order and stack on a temporary cart:

| Quantity | Height | Width |
| --- | --- | --- |
| 6 | 43½" | 96" |
| 4 | 44" | 96" |
| 7 | 38¼" | 96" |

## Vertical Cuts

Set up the panel saw for vertical cuts. For each home, cut the following
panels and mark each with the panel designator letter, and indicate the
cut edge. Stack these panels on one cart per home in this order (back to
front):

| Width | Length | Label | Source |
| --- | --- | --- | --- |
| 48" | 88" | D | Uncut panel from plywood stack |
| 38 ¼" | 46 ¼" | E | Cut two E panels from one 38¼ x 96 |
| 48" | 88" | G | Uncut panel from plywood stack |
| 38 ¼" | 88" | H | Temporary cart (thinnest panels); see below for cut-off. |
| 44" | 86 ½" | C | Temporary cart (widest panels) |
| 48" | 92" | A | Uncut panel from plywood stack |
| 38 ¼" | 92 | B | Temporary cart (thinnest panels) |
| 44" | 86 ½" | C | Temporary cart (widest panels) |
| 48" | 92" | A | Uncut panel from plywood stack |
| 38 ¼" | 92" | B | Temporary cart (thinnest panels) |
| 43 ½" | 44" | R2 | Temporary cart (middle width panels) |
| 43 ½" | 44" | R2 | Remainder from previous R2 |
| 43 ½" | 92" | R1 | Temporary cart (middle width panels) |
| 43 ½" | 92" | R1 | Temporary cart (middle width panels) |

## Table saw

The cut-off piece from the H panel above will be 38 ¼" by 8 ½".  Rip
two 4" pieces on the table saw, using the 4" Trim template. The two
halves will be used on the left side of the door, under the D
panel. Label them as "left of door" and put one on each cart. Extras
can go in the standing short-materials bin by the long-materials
shelves.
